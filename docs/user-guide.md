# Schedule For SNS ユーザー向け説明書

最終更新日: 2026年5月1日

この説明書は、Schedule For SNSを利用する方向けの案内です。

## このアプリでできること

Threadsへの投稿を、あらかじめ日時指定して予約できます。

主な機能:

- Threadsアカウントでログイン
- カレンダーから投稿日を選択
- 投稿時刻と投稿本文を設定
- 予約前に投稿日時、タイムゾーン、本文を確認
- 1日3件まで予約投稿
- 予約済み投稿の編集・削除
- 投稿済み・予約済み・失敗した投稿の確認
- 失敗した投稿から同じ本文で新しい予約を作成
- 投稿後の基本分析
- 日本語、英語、中国語、フィリピン語、ベトナム語の表示切替

## 料金

料金は税込390円/月です。

初回は14日間の無料トライアルがあります。

無料トライアル中は、管理画面に残り日数と残り時間が表示されます。

無料トライアル終了前のアプリ独自通知はありません。トライアル終了前に退会しない場合、トライアル終了後に月額390円の課金が開始されます。

領収書や支払い失敗通知などは、Stripe側からメールで送信される場合があります。

無料トライアルは同一ユーザーにつき1回のみ利用できます。退会後に再登録した場合でも、無料トライアルは再利用できません。

## ログイン

1. ログイン画面を開きます
2. 「Threadsでログイン」を選択します
3. Threads側の認証画面で連携を許可します
4. アプリに戻ると利用開始できます

Threads連携では、予約投稿と基本分析に必要な範囲の権限を利用します。

## 課金登録

1. 課金画面を開きます
2. 月額390円プランの内容を確認します
3. Stripe Checkoutに進みます
4. クレジットカード情報を入力します
5. 登録完了後、アプリに戻ります

無料トライアル中または有料プランが有効な場合に、予約投稿を作成できます。

## 予約投稿を作成する

1. メイン画面でカレンダーを開きます
2. 投稿したい日付を選択します
3. 投稿時刻を入力します
4. 投稿本文を入力します
5. 確認画面で投稿日時、タイムゾーン、投稿本文を確認します
6. 「予約する」を選択します

1日あたり予約できる投稿は3件までです。

過去の日時には予約できません。

現在時刻から5分以内の日時には予約できません。

1日の判定は、設定画面で選択したタイムゾーンを基準に行われます。

## タイムゾーン

初期設定では、ブラウザのタイムゾーンが自動で使われます。

設定画面からタイムゾーンを変更できます。

予約時刻、1日3件の制限、予約一覧の表示は、設定されたタイムゾーンを基準にします。

## 予約一覧

予約一覧では、以下の状態を確認できます。

- 予約済み
- 投稿中
- 投稿済み
- 失敗
- キャンセル済み

投稿前の予約は削除できます。

投稿前の予約は、投稿日時、タイムゾーン、投稿本文を編集できます。

投稿済みの投稿は、このアプリから削除できません。

## 投稿が失敗した場合

投稿に失敗した場合、予約一覧に失敗マークと失敗理由が表示されます。

主な失敗理由:

- Threads連携の期限切れ
- Threads再連携が必要
- Threads APIのエラー
- 投稿時刻時点で必要な権限が失効していた
- 外部サービスの一時的な障害

重複投稿を防ぐため、失敗した投稿の自動再試行は行いません。

失敗理由を確認し、必要に応じてThreads再連携または新しい予約投稿を作成してください。

失敗した投稿から、同じ本文で新しい予約を作成できます。

## Threads再連携が必要な場合

Threads連携の期限切れや権限失効が発生した場合、設定画面や予約一覧に「Threads再連携が必要」と表示されます。

この場合は、設定画面から再度Threads連携を行ってください。

再連携が完了するまで、新しい予約投稿や投稿実行が利用できない場合があります。

## 投稿分析

投稿に成功した後、以下のタイミングで基本分析を取得します。

- 投稿の1時間後
- 投稿の24時間後
- 投稿の72時間後

表示される可能性がある指標:

- 閲覧数
- いいね数
- 返信数
- リポスト数
- 引用数
- シェア数
- 合計エンゲージメント

分析指標はThreads APIから取得できる範囲で表示されます。アカウント状態やThreads APIの仕様により、一部の指標が表示されない場合があります。

## 言語設定

初回表示時は、ブラウザの言語設定をもとに表示言語が選択されます。

設定画面から以下の言語に変更できます。

- 日本語
- English
- 中文
- Filipino
- Tiếng Việt

## 支払い方法の変更

設定画面からStripe Billing Portalを開けます。

Stripe Billing Portalでできること:

- 支払い方法の変更
- サブスクリプションの確認

退会はアプリの設定画面から行います。退会するとStripeサブスクリプションも終了し、以後の定期請求は行われません。

## 退会

設定画面から退会できます。

退会時に削除される主なデータ:

- Threadsアクセストークン
- 投稿本文
- 予約投稿
- 投稿分析データ
- 通常プロフィール情報

未投稿の予約がある場合、退会時にキャンセルされます。

ただし、請求・監査・不正利用防止・無料トライアル再利用防止のために必要な最小限の情報は、復元できない識別子またはハッシュとして保持される場合があります。

無料トライアル再利用防止のため、ThreadsユーザーIDに由来する復元不能なハッシュが保持される場合があります。

## 注意事項

本アプリはThreads API、Stripe、AWSなどの外部サービスを利用しています。

外部サービスの障害、仕様変更、権限制限、API制限により、一部機能が利用できない場合があります。

障害やメンテナンスがある場合、アプリ内のバナーでお知らせすることがあります。

Threads側で投稿できない内容、またはThreadsの利用規約やMetaのポリシーに違反する内容は投稿しないでください。
